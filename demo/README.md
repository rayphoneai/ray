# Rayphoneai theme — デモサイト

> **AI × ライティング専門 WordPress テーマ「Rayphoneai theme」のデモページです。**  
> 👉 **[デモを見る](https://rayphoneai.github.io/ray/demo/)**

---

## 含まれるページ

| ファイル | 説明 |
|---------|------|
| `index.html` | トップページ（ヒーロー・記事グリッド・サイドバー） |
| `post.html` | 個別記事ページ（TOC・シェア・関連記事） |
| `assets/eyecatch-1〜6.svg` | サンプルアイキャッチ画像 |

## WordPressテーマをダウンロード

[rayphoneai-theme.zip をダウンロード](https://github.com/rayphoneai/rayphoneai-theme/releases)

## テーマの特徴

- ✅ Gemini API で記事自動生成
- ✅ Gemini Image（Nano Banana）でアイキャッチ自動生成  
- ✅ 全カラー・レイアウトを Customizer から変更可能
- ✅ 閲覧数カウンター・人気記事ウィジェット
- ✅ 自動TOC・読書進捗バー
- ✅ SEO / OGP / JSON-LD 自動出力
